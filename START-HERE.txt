┌────────────────────────────────────────────────────────────────────────────┐
│  🔍 WHAT NTIS API DOES                                                     │
└────────────────────────────────────────────────────────────────────────────┘

  • Connects to official Korean government R&D database
  • Searches 108,798+ active R&D funding programs
  • Returns structured XML data (no HTML parsing needed!)
  • Searches by:
    ✓ Recent announcements (last N days)
    ✓ Keywords
    ✓ Agency (IITP, TIPA, KIMST, KEIT)
    ✓ Custom filters
  • Automatic deduplication in database
  • Rate limiting built-in (10 requests/min)
